# DVWA Cyber Security Pipeline – Learning Project

This repository is a **learning and experimentation project** built around the
*Damn Vulnerable Web Application (DVWA)* codebase.

## Purpose
- Cyber‑security practice
- CI/CD & security pipeline learning
- Vulnerability testing in controlled environments

## Important Notice
This project contains **intentionally vulnerable code**.
⚠️ **Do NOT deploy publicly.**
Use only in local labs, VMs, or isolated environments.

## Attribution
This project is based on the open‑source **DVWA** project originally created by
the DVWA team (digininja).

The repository has been cleaned and restructured for **educational and pipeline‑learning purposes**.
